Commands:-
pwd                       - it shows current working directory
ls                        - it shows the files in the directory
ls-la                     - it shows the files in the permissions format and order wise 
cd                        - change directory
cd ..                     - to go to one step back from the current directory to previous directory or files 
mkdir                     - to create directories
touch (file.txt)          - it creates files 
cp (file.txt file1.txt)   - it copies the data from one file to another file
mv (file1.txt moved.txt)  - this commmand is used to move or rename the file or directory
rm moved.txt              - this command is used to remove the files or directory
clear                     - clear the screen
history                   - it shows the commands used from beginning
