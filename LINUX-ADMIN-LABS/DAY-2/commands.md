cat  file.txt           - open the file
nano file.txt           - edit the file with text editor
head file.txt           - first 10 lines are printed on screen
tail file.txt           - last 10 lines are printed on screen
find . -name "*.txt"    - find the file using path,extension,types etc.,
locate file.txt         - print the file location
wc -l  file.txt         - word count of the file
sort   file.txt         - order the file data
